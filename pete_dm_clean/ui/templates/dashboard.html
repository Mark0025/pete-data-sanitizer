TL;DR (how to use this UI)

  1. Create or select a Client. This keeps each client’s files and runs separate.
  2. Upload the inputs for that client:
    • desired_outcome CSV
    • contacts CSV
    • template XLSX (optional; defaults to Properties Template (15))
  3. Build: choose your export name settings, optionally enable debug report + deep diagram, then click Run build.
Tip: if you select “(auto)” in Build, the app only auto-detects files inside the selected client. If that client has no uploads yet, Build will tell you to upload/select inputs first.
If you just created a new client and haven’t uploaded files yet, “(auto)” will fall back to the shared uploads so you can still run a build. Best practice is to upload the files into the client workspace.
